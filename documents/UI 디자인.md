# [UI/UX 설계서] StudyFlow: 목차 기반 지능형 학습 스케줄러

---

## 2. 페이지별 상세 기획

### 2.1 랜딩 페이지 (Landing: Introduction)

수익 창출을 위한 광고성 문구를 제거하고, 서비스의 본질적인 가치와 사용법을 소개하는 데 집중합니다.

- **헤더(Navigation)**: 좌측 로고, 우측 [소개 / 사용법 / 로그인] 버튼.
- **히어로 섹션(Hero)**:
  - 중앙 정렬된 Serif 체 헤드라인: "공부의 구조를 잡다, 스케줄을 자동화하다."
  - 서비스를 한눈에 보여주는 고해상도 대시보드 스크린샷 또는 목업 이미지.
- **기능 소개 섹션**:
  - **OCR**: "이미지 한 장으로 목차를 텍스트로."
  - **Editor**: "직접 추가하고 계층을 조절하는 자유로운 편집."
  - **Auto-Reschedule**: "어제 못한 공부는 오늘로, 자동으로 밀려오는 유연함."
- **하단**: 간단한 프로젝트 소개 및 푸터.

### 2.2 로그인 페이지 (Login)

간결하고 전문적인 폼을 제공합니다.

- **중앙 카드 레이아웃**: 깨끗한 화이트 배경의 카드.
- **기본 로그인**: 이메일과 비밀번호 입력창 (1px 테두리의 미니멀한 스타일).
- **간편 로그인**:
  - **카카오톡 로그인**: 카카오 고유 컬러를 유지하되 전체적인 톤앤매너를 해치지 않는 깔끔한 버튼 디자인.
- **하단**: [계정 만들기] 링크.

### 2.3 스마트 목차 편집기 (TOC Editor)

이미지와 텍스트 리스트가 공존하는 정밀한 작업 공간입니다.

- **Split View**:
  - **좌측**: 업로드한 목차 이미지 고정 뷰어 (줌 기능 포함).
  - **우측**: 추출된 텍스트 및 수동 추가 항목 리스트.
- **항목 제어 (Row Component)**:
  - 항목 선택 시 미세한 주황색 보더 강조.
  - 들여쓰기(Depth)는 버튼 대신 가느다란 점선(Guide Line)으로 시각화.
  - 각 행마다 [수정 / 삭제 / 삽입(+)] 기능 제공.
- **기능 버튼**: 하단 혹은 우측 상단에 [스케줄 생성 알고리즘 실행] 버튼 (주황색 강조).

### 2.4 목표 설정 모달 (Goal Setup)

사용자의 학습 의지를 수치화하는 단계입니다.

- **모드 선택**: '목표 날짜 지정' 또는 '일일 학습량 지정' (깔끔한 세그먼트 버튼).
- **입력 요소**: 세리프 숫자가 적용된 정교한 넘버 피커 및 미니멀 캘린더 위젯.
- **유효성 검사**: Zod를 사용하여 필수 항목 누락 시 주황색 텍스트로 에러 표시.

### 2.5 메인 대시보드 (Daily Dashboard)

학습 현황을 모니터링하고 관리하는 중앙 통제 센터입니다.

- **상단 상태 바**: 전체 진행률(Progress Bar), 남은 일수, 완료한 항목 수 등을 벤토 박스(Bento Box) 스타일로 배치.
- **날짜 리본**: 상단에 가로로 나열된 날짜들 중 오늘 날짜를 얇은 주황색 원으로 강조.
- **할 일 리스트**:
  - 오늘의 할 일 항목들.
  - 체크박스는 원형의 미니멀한 디자인.
  - 항목 완료 시 텍스트에 옅은 회색 취소선 적용.
- **자동 밀기 알림**: 미완료 항목 존재 시 대시보드 최상단에 "2개의 항목이 이월되었습니다."라는 절제된 안내 문구 노출.

## 3. 핵심 인터랙션 및 로직

1.  **계층형 데이터 관리**: Zustand를 사용하여 사용자가 들여쓰기를 수정할 때마다 실시간으로 `depth` 값 동기화.
2.  **Zod 검증**:
    - 이메일 형식, 비밀번호 최소 길이 검사.
    - 목차 저장 전 빈 항목 체크.
3.  **Auto-Reschedule**:
    - 매일 최초 접속 시, 어제 날짜까지 완료되지 않은 `Schedule` 항목을 조회.
    - 미완료 항목의 `assigned_date`를 오늘로 변경하고, 그 이후 모든 일정의 날짜를 자동으로 하루씩 Shift.
4.  **반응형 대응**: 편집기는 데스크톱 뷰를 기본으로 하며, 대시보드와 로그인은 모바일 최적화 레이아웃 제공.
